# Head Pong
24시간 동안 제작한 FPS 게임

### 스토리
사방에서 몰려오는 인형 좀비들을 쓰러트리고 살아남아라!

### 조작키
* **B** : 상점
* **W, A, S, D** : 이동키
* **좌 클릭** : 공격
* **우 클릭** : 줌인
